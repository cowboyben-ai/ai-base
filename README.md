# AI 素養養成課程 - Landing Page

這是一個專為「AI 素養養成課程」設計的高級感單頁式 Landing Page。本網頁整合了 AI 時代的趨勢分析、核心教學理念、實戰創作方法以及詳細的課程方案。

## 🚀 專案特點

- **現代化設計**：採用深色模式 (Dark Mode) 與玻璃擬態 (Glassmorphism) 視覺效果。
- **響應式排版**：適配各種螢幕尺寸，提供極佳的閱讀體驗。
- **豐富視覺內容**：包含專屬生成的 AI 輔助視覺圖標與場景圖。
- **動態效果**：整合了滾動顯示 (Scroll Reveal) 動畫，增加互動感。

## 📚 課程核心

網頁內容基於「AI 素養」三大核心能力：
1. **表達想法**：將模糊想法轉化為清楚的 AI 指令。
2. **完成創作**：透過 AI 協作，專注於思維而非技術門檻。
3. **檢查與判斷**：建立獨立思考，決定 AI 產出的最終品質。

教學架構包含 **Artifact 創作法** 與 **LobeHub 互動教學**，引導孩子經歷「提問 → 生成 → 檢查 → 修改 → 發表 → 反思」的完整循環。

## 🛠️ 技術棧

- HTML5
- Vanilla CSS (自定義變量 + 現代排版)
- Google Fonts (Inter, Outfit, Noto Sans TC)
